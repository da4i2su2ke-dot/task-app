# タスク管理アプリ

## 概要
タスクの作成・管理・削除ができるWebアプリです。

## 使用技術
- Java 17
- Spring Boot
- Thymeleaf
- H2 Database
- JUnit
- GitHub Actions

## なぜこのアプリを作ったか
日常のタスク管理をメモ帳で行っていたが、完了したタスクを消してしまうと
履歴が残らず振り返りができなかった。
ステータスで管理することで履歴を残しつつ進捗を把握できるアプリを作りたいと思い開発した。

## 工夫した点
タスクに「未着手・進行中・完了」のステータスを持たせることで、
削除せずに進捗を管理できるようにした。
